CustomGPT Isolated Widgets Example

About Conversation Isolation

By default, each widget instance has isolated conversations. This means:

  • The embedded widget and floating button have separate conversation histories
  • Messages in one widget don't appear in the other
  • Each widget maintains its own set of conversations

To share conversations between widgets, set isolateConversations: false and provide the same sessionId.

1. Embedded Widget (Isolated)

This widget has its own conversation history, separate from the floating button.

2. Floating Button (Isolated)

Click the floating button in the bottom-right corner. It has its own separate conversations.

3. Shared Conversation Example

If you want widgets to share conversations, use the same sessionId:

// Both widgets will share conversations
const sharedSessionId = 'shared-session-123';

CustomGPTWidget.init({
    apiKey: 'your-api-key',
    agentId: '123',
    sessionId: sharedSessionId,
    isolateConversations: false,  // Important!
    // ... other config
});

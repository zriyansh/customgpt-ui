CustomGPT Widget - Vanilla JavaScript Integration

1. Basic Embedded Widget

Simple embedded chat widget with default settings.

// Basic initialization const widget = CustomGPTWidget.init({ apiKey: 'your-api-key', agentId: '123', containerId: 'chat-container', mode: 'embedded' });

2. Widget with Conversation Management

Full-featured widget with conversation switching, limits, and persistence.

// With conversation management const widget = CustomGPTWidget.init({ apiKey: 'your-api-key', agentId: '123', agentName: 'Support Bot', containerId: 'managed-chat', mode: 'embedded', enableConversationManagement: true, maxConversations: 5, onConversationChange: (conv) => { console.log('Switched to:', conv.title); } });

3. Floating Button Widget

Floating chat button that appears in the corner of the page.

// Floating button const floatingWidget = CustomGPTWidget.init({ apiKey: 'your-api-key', agentId: '123', agentName: 'Assistant', mode: 'floating', position: 'bottom-right', enableConversationManagement: true, maxConversations: 3 });
Look for the floating button in the bottom-right corner after initialization.

4. Advanced API Usage

Demonstrates programmatic control of the widget.

// Get all conversations const conversations = widget.getConversations(); // Create new conversation const newConv = widget.createConversation('Order Support'); // Switch conversation widget.switchConversation(conversationId); // Update title widget.updateConversationTitle(conversationId, 'New Title'); // Delete conversation widget.deleteConversation(conversationId); // Force refresh widget.refresh(); 

            

        


        
        

            
5. Multiple Widget Instances

            
Run multiple chat widgets on the same page with different configurations.

            
            

// Sales chat
const salesWidget = CustomGPTWidget.init({
    apiKey: 'your-api-key',
    agentId: 'sales-agent',
    agentName: 'Sales Bot',
    containerId: 'sales-chat',
    sessionId: 'sales-session' // Different session
});

// Support chat
const supportWidget = CustomGPTWidget.init({
    apiKey: 'your-api-key',
    agentId: 'support-agent',
    agentName: 'Support Bot',
    containerId: 'support-chat',
    sessionId: 'support-session' // Different session
});

6. CDN Integration

            
Load the widget from CDN for easy integration.

            
            

<!-- Include from CDN (vendors.js must be loaded first!) -->
<script src="https://cdn.customgpt.ai/widget/latest/vendors.js"></script>
<script src="https://cdn.customgpt.ai/widget/latest/customgpt-widget.js"></script>
<link rel="stylesheet" href="https://cdn.customgpt.ai/widget/latest/customgpt-widget.css">

<!-- Initialize widget -->
<script>
  CustomGPTWidget.init({
    apiKey: 'your-api-key',
    agentId: '123',
    containerId: 'chat',
    mode: 'embedded'
  });
</script>
            

            
            

                Note: Replace CDN URLs with actual CustomGPT CDN endpoints when available.

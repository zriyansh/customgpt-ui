Basic Usage

Include the iframe embed script and initialize the widget:

<!-- Include the iframe embed script -->
<script src="./dist/widget/iframe-embed.js"></script>

<!-- Create a container for embedded mode -->
<div id="my-chat-widget"></div>

<!-- Initialize the widget -->
<script>
    const widget = CustomGPTEmbed.init({
        apiKey: 'your-api-key-here',
        mode: 'embedded',
        containerId: 'my-chat-widget',
        iframeSrc: 'https://your-domain.com/iframe/',
        onReady: () => console.log('Widget ready'),
        onMessage: (data) => console.log('New message:', data)
    });
</script>

Configuration Options

Option Type Default Description
apiKey string required Your CustomGPT API key
mode 'embedded' | 'floating' 'embedded' Widget display mode
containerId string undefined Container element ID (required for embedded mode)
iframeSrc string 'https://your-domain.com/widget/' URL of the iframe source
position 'bottom-right' | 'bottom-left' | 'top-right' | 'top-left' 'bottom-right' Position for floating mode
width string '400px' Widget width
height string '600px' Widget height
zIndex number 9999 CSS z-index for floating mode
boxShadow string '0 10px 25px rgba(0, 0, 0, 0.2)' CSS box shadow
borderRadius string '12px' CSS border radius

Event Callbacks

Callback Parameters Description
onReady - Called when the widget iframe is ready
onOpen - Called when the widget is opened
onClose - Called when the widget is closed
onMessage message Called when a new message is sent or received
onError error Called when an error occurs

Widget Methods

Method Parameters Description
widget.open() - Open the widget (floating mode)
widget.close() - Close the widget (floating mode)
widget.toggle() - Toggle widget visibility
widget.destroy() - Remove the widget from the page
widget.updateConfig() newConfig Update widget configuration
widget.isOpened - Check if widget is open (getter)
widget.configuration - Get current configuration (getter)

Advanced Example

<script>
    // Initialize with full configuration
    const advancedWidget = CustomGPTEmbed.init({
        apiKey: 'your-api-key-here',
        mode: 'floating',
        position: 'bottom-right',
        width: '450px',
        height: '650px',
        theme: 'light',
        enableCitations: true,
        enableFeedback: true,
        zIndex: 10000,
        
        // Event handlers
        onReady: () => {
            console.log('Widget is ready!');
            updateStatus('Widget initialized successfully', 'success');
        },
        
        onOpen: () => {
            console.log('Widget opened');
            // Track widget opening event
            analytics.track('widget_opened');
        },
        
        onClose: () => {
            console.log('Widget closed');
        },
        
        onMessage: (message) => {
            console.log('New message:', message);
            // Handle message events
            if (message.type === 'user_message') {
                // Track user engagement
                analytics.track('user_message_sent');
            }
        },
        
        onError: (error) => {
            console.error('Widget error:', error);
            updateStatus('Widget error: ' + error.message, 'error');
        }
    });
    
    // Programmatic control
    setTimeout(() => {
        advancedWidget.open();
    }, 2000);
</script>

Security Considerations

  • Origin Verification: The iframe can verify the parent window origin for additional security
  • Content Security Policy: Configure CSP headers to control iframe embedding permissions
  • Sandbox Attributes: The iframe uses sandbox attributes to restrict potentially harmful operations
  • Message Validation: All postMessage communications include type validation
  • HTTPS Required: Always serve the iframe content over HTTPS in production

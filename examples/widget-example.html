CustomGPT Widget Integration Examples

1. Embedded Widget

The widget is embedded directly in a container on your page.

<div id="customgpt-embedded" style="width: 100%; height: 600px;"></div>
<script src="https://cdn.customgpt.ai/widget/customgpt-widget.js"></script>
<script>
  CustomGPTWidget.init({
    apiKey: 'YOUR_API_KEY',
    agentId: 123, // Your agent ID
    containerId: 'customgpt-embedded',
    mode: 'embedded'
  });
</script>

2. Floating Widget

The widget appears as a chat bubble that can be opened/closed.

<script src="https://cdn.customgpt.ai/widget/customgpt-widget.js"></script>
<script>
  const floatingWidget = CustomGPTWidget.init({
    apiKey: 'YOUR_API_KEY',
    agentId: 123, // Your agent ID
    mode: 'floating',
    position: 'bottom-right'
  });
</script>

Widget Controls

You can programmatically control the widget using JavaScript.

// Open the widget
widget.open();

// Close the widget
widget.close();

// Toggle the widget
widget.toggle();

// Update configuration
widget.updateConfig({
  theme: 'dark'
});

// Destroy the widget
widget.destroy();

Configuration Options

All available configuration options for the widget:

{
  apiKey: 'YOUR_API_KEY',      // Required: Your CustomGPT API key
  agentId: 123,                 // Required: Your agent/project ID
  containerId: 'container-id',  // For embedded mode: container element ID
  mode: 'embedded',             // 'embedded' | 'floating' | 'widget'
  theme: 'light',               // 'light' | 'dark'
  position: 'bottom-right',     // For floating: 'bottom-right' | 'bottom-left' | 'top-right' | 'top-left'
  width: '400px',               // Widget width
  height: '600px',              // Widget height
  enableCitations: true,        // Show citation sources
  enableFeedback: true,         // Show feedback buttons
  onOpen: () => {},             // Callback when widget opens
  onClose: () => {},            // Callback when widget closes
  onMessage: (msg) => {}        // Callback when message is sent/received
}

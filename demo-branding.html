Customization Options

The branding is built into the components, but you can customize it by modifying the style properties in the source code:

  • Font Size: Adjust the fontSize property (currently 11px for widget, 10px for floating)
  • Color: Change the color property (currently #999)
  • Background: Modify the backgroundColor (currently #fafafa)
  • Padding: Adjust spacing with the padding property
  • Border: Customize the borderTop style
